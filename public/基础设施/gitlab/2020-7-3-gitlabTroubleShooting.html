# 【解决】：** .gitlab-ci.yml中增加tags：与runner的tags对应上。

# 【问题】【gitlab runner registe error】

# [现象]

在runner机器上，docker exec -it gitlab_runner gitlab-runner register ,填写tag的时候就僵住不动， 不一会就报error。说连不上gitci服务器了。ping GitCi服务器都能ping通。

# [解决] docker 重启

systemctl restart docker 重新关掉容器docker stop xxx && docker rm xxx，然后docker-compose up -d重启容器
