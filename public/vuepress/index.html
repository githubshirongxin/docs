# Step1, image生成目录

/docs/.vuepress/public/images/ 修改paste image插件的配置

  • path ${projectRoot}/docs/.vuepress/public/images
  • base path ${projectRoot}/docs/.vuepress/public/images

# Step2, md中ctrl+alt+v 生成的代码

  • insert pattern ${imageSyntaxPrefix}/docs/images/${imageFilePath}${imageSyntaxSuffix}
