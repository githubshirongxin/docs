vscode进入container，开发container里的文件。不用虚拟机，开发完就扔掉。程序可以持久化到本地。 不用安装hyperv，下一个docker，docker pull centos就拥有了70M大小的linux。

启动起来。

λ docker container run  --name=mycentos --volume=/data:/data/  --restart=always  -d centos

然后ctl+shift+p：remote-containers：Attach to Running Container

然后，选择一个container，进入。 就像在本地一样，进入container，开发container里的文件。

这些container是在vscode所在机器上的容器。
