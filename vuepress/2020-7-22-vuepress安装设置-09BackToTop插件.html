# 9 Back TO Top插件

插件→back to top

# 安装

npm install -D @vuepress/plugin-back-to-top

# 使用

// .vuepress/config.js
module.exports = {
  plugins: ['@vuepress/back-to-top']
}

# 效果

在页面最下面有个向上的箭头，点击就会回到顶 相当与网页按了“CTL+HOME”
