## 如何创建自己的知识库。
1. git clone 下载任意知识库代码
2. 在gitlab.ccbjb.com.cn上创建一个组（注意不要有中横线，下横线，空格，汉字）
3. 创建一个项目名docs（必须是docs，否则需要修改配置文件）
4. 把第1步的资源提交到第三步的项目中。
5. 查看第三步的项目的CICD是否运行。
