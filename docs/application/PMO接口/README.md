- PMO系统被动接受数据请求，返回json数据。
- json数据包括，人员一览和人员项目一览。项目结构。部门结构。
- 他系统随时调用PMO系统的api接口获得数据。

- 认证信息放到请求包。账号，密码新创建一个账号。他系统统一管理账号和密码，更新。


## 1. 人员（PMO上全字段）
- request
  全项目组，全人员
```json
  {
      "日期":""
  }
```
- resposne 
```json
[
    {"姓名":"",
    "工号":"",
    "采用区分":"",
     字段按PMO现有字段来
    }
]
```
